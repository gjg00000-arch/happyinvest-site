환불 요청 · USD 송금

정규 플랜 사용자가 접수한 이유 불문 일할 환불 요청입니다. 이벤트·쿠폰·무료체험·리커버리는 환불불가입니다. 송금 후 참조번호를 입력하고 완료 처리하세요.

비즈니스(조직) · 좌석 ID (TRV / MT5)

TRV(TradingView)는 차트·스크립트에 쓰는 사용자명을, MT5는 계정 로그인(일반적으로 숫자)과 브로커에서 받은 서버 이름을 적습니다(예: MetaQuotes-Demo 등). API는 GET/POST /api/admin/business/orgs·PATCH /api/admin/business/orgs/:id/seats/:seatIndex입니다. 좌석 저장 시 BUSINESS_SEAT_NOTIFY_EMAIL가 없으면 ADMIN_EMAIL로 변경 알림(SMTP 설정 시)이 갑니다.
