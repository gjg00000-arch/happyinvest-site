MetaTrader 5

지표 사용 — MT5

배포되는 .ex5는 틱(Tick) 차트 전용입니다. M1·M5 등 일반 시간봉 전용 MT5 패키지는 판매·배포하지 않습니다. 파일을 터미널이 읽는 폴더에 넣고, 틱 차트에 올린 뒤 읽습니다. 브라우저 TV만 쓰시면 지표 사용 — TradingView를 보세요.

목차

  1. 파일 설치
  2. 틱(Tick) 차트에서 쓰기
  3. 라이선스(계좌+서버)
  4. 게스트 체험(회원가입 없이·웹훅)
  5. 읽는 기본(공통)

파일 설치

  1. 브로커 제공 MT5를 설치합니다.
  2. 배포받은 .ex5를 MQL5\Indicators (또는 운영 안내 경로)에 둡니다. 파일 → 데이터 폴더 열기로 이동할 수 있어요.
  3. 터미널을 재시작하거나 내비게이터를 새로고침한 뒤, 사용자 지표에 나타나는지 확인합니다.
  4. 틱 차트를 연 뒤 끌어다 놓거나 삽입 → 지표 → 사용자 지표에서 선택합니다.

자세한 경로는 가이드 · MT5, 제품 표는 다운로드입니다.

틱(Tick) 차트에서 쓰기

제공 지표는 거래 발생 단위를 축으로 그리는 틱 차트(또는 터미널이 제공하는 동일 목적 차트 종류)에 맞춰 빌드됩니다. 시간봉만 붙여서 해석하면 의도와 다르게 보일 수 있으니, 다운로드 · MT5 틱 전용 패키지와 공지 기준으로 환경을 맞춥니다.

접근권·파일 이름에 _Tick 등이 붙을 수 있습니다. 시간봉 전용 패키지는 별도로 판매하지 않습니다.

라이선스(계좌+서버)

운영 정책이 MT5 계좌 번호+서버 1조합 기준이라면, 회원·연동에 입력한 조합이 서버·라이선스와 맞아야 합니다. 변경·오류는 연동 — MT5와 문의로 연결합니다. 브로커명만으로는 식별하지 않는 경우가 많습니다.

게스트 체험(회원가입 없이·웹훅)

일부 MT5 빌드는 준회원/정회원 가입 없이 체험할 수 있도록, 터미널에서 서버 API로 mt5_login·mt5_server 조합을 보내 시작 시각부터 기간(예: 7일·30일)을 원장에 남길 수 있습니다. 회원 플랜 원장과 같은 subject_key로 충돌하지 않으며, 이미 회원이 해당 계좌로 활성화한 경우 게스트 등록은 거절됩니다.

  • 엔드포인트(예시): POST …/api/webhooks/mt5-guest/trial-register — 운영 API 베이스 URL은 호스트 안내를 따르세요.
  • 헤더: X-Webhook-Secret — 서버에 설정된 게스트 전용 시크릿(시그널 웹훅과 다른 값 권장).
  • JSON 바디: mt5_login, mt5_server(필수) · 선택 trial_days(예: 7, 30, 서버 허용 목록에 없으면 기본값) · 선택 product(제품 라벨).
  • 이후 EA는 기존과 같이 POST /api/entitlement/magictrading/verify로 권한을 확인합니다.

약관·무료체험 특약은 제품별 배포 안내를 따르고, 게스트 경로는 남용 방지를 위해 시크릿·IP 기록 등 운영 정책이 함께 적용됩니다.

읽는 기본(공통)

지표는 수익을 보장하지 않는 참고선으로 쓰는 것이 전제입니다.

  1. 한 화면에 지표·오브젝트를 과하게 쌓지 않고, 한 심볼·틱 차트 규모부터 익혀 갑니다.
  2. 뉴스·갭·유동성이 무너지는 구간은 체크리스트를 따로 둡니다.
