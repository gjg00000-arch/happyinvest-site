MetaTrader 5

지표 사용 — MT5

.ex5 를 터미널이 읽는 폴더에 넣고, 캔들·봉용과 틱용 빌드를 섞지 않게 쓰는 것이 첫걸음입니다. TRV(브라우저)만 쓰시는 경우는 지표 사용 — TradingView를 보세요.

목차

  1. 파일 설치
  2. 캔들용 vs 틱용
  3. 라이선스(계좌+서버)
  4. 읽는 기본(공통)

파일 설치

  1. 브로커 제공 MT5를 설치합니다.
  2. 배포받은 .ex5를 MQL5\Indicators (또는 운영 안내 경로)에 둡니다. 파일 → 데이터 폴더 열기로 이동할 수 있어요.
  3. 터미널을 재시작하거나 내비게이터를 새로고침한 뒤, 사용자 지표에 나타나는지 확인합니다.
  4. 차트에 끌어다 놓거나 삽입 → 지표 → 사용자 지표에서 선택합니다.

자세한 경로는 가이드 · MT5, 제품 표는 다운로드입니다.

캔들용 vs 틱용

다운로드 안내에 일반(캔들·봉)과 Tick이 나뉘어 있으면, 틱 전용은 틱 차트/틱봇 환경에 맞게 써야 읽기가 흐트러지지 않습니다. M1·M5 등 봉 주기를 먼저 정한 뒤 지표를 겁니다.

틱 차트용 빌드 구분: 다운로드 · Tick과 모임터 공지를 함께 보세요.

라이선스(계좌+서버)

운영 정책이 MT5 계좌 번호+서버 1조합 기준이라면, 회원·연동에 입력한 조합이 서버·라이선스와 맞아야 합니다. 변경·오류는 연동 — MT5와 문의로 연결합니다. 브로커명만으로는 식별하지 않는 경우가 많습니다.

읽는 기본(공통)

지표는 수익을 보장하지 않는 참고선으로 쓰는 것이 전제입니다.

  1. 한 화면에 지표·오브젝트를 과하게 쌓지 않고, 한두 심볼·한 타임프레임부터 익혀 갑니다.
  2. 뉴스·갭·유동성이 무너지는 구간은 체크리스트를 따로 둡니다.
