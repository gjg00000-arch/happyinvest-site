이 사이트에서 할 수 있는 것

가이드·가입·모임터를 한곳에 모아 두었습니다. 지표는 수익을 보장하는 상품이 아니라 차트를 읽는 참고 도구입니다. 플랜·혜택은 이벤트·회원 혜택에서, 게시판 권한은 접근 권한 표를 참고하세요.

TRV MT5 Telegram Webhook

회원 가입 · 본인 인증 · 결제

대략적인 순서입니다. 화면은 배포 시점에 따라 달라질 수 있어요.

  1. 가입·등록에서 계정을 만듭니다.
  2. 본인 인증으로 계정을 보호합니다. (문자 발신 등은 운영 정책에 따름)
  3. 구독·결제에서 플랜을 선택합니다. 연동 전에는 공지를 따라 주세요.
  4. 로그인 후 MT5 계좌 · MQL5 · TRV ID를 프로필에 연결하면 라이선스·알림 매칭에 쓸 수 있습니다.

TRV에 지표 넣기

TV에서는 보통 초대 링크 또는 스크립트 공개 형태로 지표를 받습니다. 차트 상단 Indicators에서 커뮤니티·내 스크립트를 확인하거나, 제공된 링크를 눌러 추가합니다.

  1. TRV에서 차트를 엽니다.
  2. 배포 방식에 따라 Pine 스크립트를 추가하거나, 초대 링크를 통해 지표를 불러옵니다.
  3. 지표 설정(기간·색 등)은 속성 창에서 조정합니다. 플랜에 따라 알림·스크린 수 제한이 있을 수 있습니다.

MQL5 마켓과 달리 TV는 브라우저·앱 안에서 동작합니다. mql5.com 로그인 이메일은 회원 정보에 넣어 두면 나중에 라이선스·문의 매칭에 도움이 됩니다.

MT5 — 파일 받기 · 넣기 · 틱 차트에 붙이기

순서는 ① 파일을 손에 넣고 → ② 터미널이 목록을 읽고 → ③ 틱 차트를 연 뒤 지표 넣기입니다. 배포 .ex5는 틱 차트 환경 전용입니다.

  1. 브로커에서 제공하는 MetaTrader 5를 설치합니다.
  2. 받은 .ex5 파일을 MQL5\Indicators 폴더에 넣습니다. (메뉴 파일 → 데이터 폴더 열기로 바로 갈 수 있습니다.)
  3. MT5를 한 번 재시작하거나 내비게이터를 새로고침하면, 사용자 지표 목록에 나타납니다.
  4. 틱(Tick) 차트를 연 다음, 끌어 놓거나 삽입 → 지표 → 사용자 지표에서 선택합니다.

원본 .mq5는 보통 배포하지 않고, 컴파일된 .ex5만 쓰는 것이 일반적입니다. 브로커·OS마다 경로 표기가 조금씩 다를 수 있습니다.

텔레그램 Chat ID 확인

알림·웹훅을 보내려면 내 채팅 ID가 필요합니다. 봇과 대화를 시작한 뒤, BotFather로 만든 봇 토큰과 함께 getUpdates 등으로 숫자 ID를 확인하는 방식이 흔합니다.

  1. 안내된 봇을 채팅방에 추가하거나 1:1 대화를 시작합니다.
  2. 개인 채팅과 그룹·채널은 ID 형식이 다릅니다. 운영 안내에 맞는 쪽으로만 등록해 주세요.
  3. ID는 타인에게 불필요하게 노출하지 않는 것이 좋습니다.

서버·웹훅 설계 개요는 연동 페이지를 참고하세요. 국내 SMS 본인인증은 별도 발신번호 등록이 필요합니다 — 본인인증 안내와 함께 정리합니다.

시그널 웹훅 (선택)

magic-indicator-api를 쓰는 경우, TRV·MT5 알림을 HTTP로 받아 로그를 남기고 텔레그램으로 전달할 수 있습니다. TradingView Webhook URL·JSON 필드 표·정적 사이트와의 구분은 연동 — TRV 시그널 웹훅에 모아 두었습니다. API README와 함께 보시면 됩니다.

  • 엔드포인트 예: POST /api/webhooks/signals/tradingview, POST /api/webhooks/signals/mt5 — 상태 확인은 GET …/api/webhooks/signals/health
  • 인증: 헤더 X-Webhook-Secret, Authorization: Bearer …, 또는 쿼리 ?secret= (서버 환경변수와 동일 값)
  • 본문은 JSON·텍스트 등 raw 수신을 전제로 합니다. 응답의 telegram.* 필드로 전달 결과를 확인합니다.

경제 지표(고임팩트) 텔레그램 사전 알림은 TRV 시그널과 별도입니다. Finnhub 캘린더를 당겨오고, 스케줄러가 POST /v1/cron/economic-calendar로 백엔드를 깨우는 방식입니다. 절차·Docker·Nginx·curl 예시는 연동 — 경제 지표 Cron 웹훅과 저장소 server/economic-cron/DEPLOY.md를 보세요.

매직라인 운용 팁 (요약)

틱 차트·파생 상품 전제에 가깝게 쓰일 수 있고, R값으로 신호 빈도를 조절합니다. 완벽한 지표는 없으니 손절·자금 관리는 별도로 지켜 주세요. 메인 페이지의 안내 문구도 함께 읽어 주시면 좋습니다.
