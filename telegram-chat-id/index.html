텔레그램 Chat ID 확인

가입 폼의 텔레그램 Chat ID란에는 숫자만 넣습니다. @username 과는 다른 값이에요. 아래 확인란을 체크하면 방법 보기 버튼이 켜지고, 누르면 단계별 안내가 펼쳐집니다.

먼저 잠깐만 읽어 주세요

위 확인란을 체크해야 버튼이 활성화됩니다.

단계별 안내

  1. 텔레그램 검색창에서 @userinfobot을 검색해 입장합니다.
  2. 하단의 [시작] 버튼을 누르거나 /start를 입력합니다.
  3. 봇이 즉시 보내는 답장에서 Id 항목의 숫자를 확인합니다. 이 숫자를 회원 가입의 텔레그램 Chat ID (선택) 칸에 넣으면 됩니다.

Chat ID는 숫자값이며, @username과 다릅니다. 불필요한 공개는 피하세요.
