Adding the indicator on TRV

TV delivery is usually via invite links or published Pine scripts. Open Indicators on the chart, browse community or invite links, then add what we provide.

  1. Open a chart on TRV.
  2. Add the Pine script per release notes, or tap the invite URL we share.
  3. Tune lengths and colors in Properties; alerts and screen quotas depend on plan.

Unlike the MQL5 Market, TradingView runs in the browser or app. Saving your mql5.com login email in membership details helps reconcile licences later.

MT5 — download, deploy, attach on tick charts

Flow: ① Drop the compiled file → ② refresh MT’s lists → ③ Open a tick chart and attach. Shipments are tick-chart-only .ex5.

  1. Install MetaTrader 5 from your broker.
  2. Copy the bundled .ex5 into MQL5\Indicators (use File → Open Data Folder).
  3. Restart MT or refresh Navigator—the indicator appears under Custom Indicators.
  4. Open a Tick chart first, drag the indicator onto it, or use Insert → Indicators → Custom.

Sources .mq5 are typically not shipped; brokers and OS locales may tweak folder paths slightly.

Telegram Chat ID checklist

Alerts/webhooks usually need your numeric chat ID. Invite the bot per instructions, combine its token with getUpdates helpers, then store only the authorised channel/group ID.

  1. Add the announced bot or start its DM conversation.
  2. DM, group and channel IDs differ—register only where operations say it is permitted.
  3. Do not expose IDs casually.

Architecture notes live under Integrations. KR SMS OTP may require outbound sender IDs—follow Verification notices.

Signal webhooks (optional)

With magic-indicator-api you may capture TRV/MT5 payloads over HTTP and fan them out—URL templates, JSON hints, and how this differs from the static brochure site are spelled out under TRV webhook integration alongside API README excerpts.

  • Typical endpoints include POST /api/webhooks/signals/tradingview, POST /api/webhooks/signals/mt5; health probes use GET …/api/webhooks/signals/health.
  • Auth variants: headers X-Webhook-Secret, Authorization: Bearer …, or ?secret= matching server env vars.
  • Payloads arrive as JSON or plaintext; Telegram delivery status often surfaces inside telegram.* fields.

High-impact FX calendar pings are separate from TRV signals—they fetch Finnhub, then a scheduler calls POST /v1/cron/economic-calendar. Scripts, Docker, Nginx snippets: see Economic Cron webhook and server/economic-cron/DEPLOY.md.
